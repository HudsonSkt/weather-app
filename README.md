# 🌤 Weather App

## 📌 Sobre o Projeto
- Weather App é um App criado para consultas de clima de qualquer cidade, estado ou país disponível , com um incrível visual Glassmorphism.

## 🛠️ Tecnologias Utilizadas
- React.js
- TypeScript
- Vite
- Styled Components
- Axios
- OpenWeatherMap API

## 🔗 Funcionalidades
- Busca de clima por nome da cidade
- Exibe temperatura atual e sensação térmica
- Exibe descrição do clima, umidade e velocidade do vento
- Ícone dinâmico de acordo com o clima
- Suporte ao Enter para buscar

## ▶️ Como Executar
```bash
# Clone o repositório
git clone https://github.com/HudsonSkt/weather-app

# Instale as dependências
npm install

# Crie o arquivo .env baseado no .env.example
# e adicione sua chave da OpenWeatherMap

# Rode o projeto
npm run dev
```
